## [1.1.2] - 2026-05

视觉统一升级 + 自动回复流程修复。

### 新增

- **三页视觉统一升级**：主动寻访 / 自动回复 / 历史三个 tab 套用同一套设计语言 —— 玻璃质感卡片、活动 tab 顶部高光线、紫蓝渐变标题、纤细滚动条、微金属感状态栏
- **每页统一节奏小标头**：每个 tab 顶部加 `· 标题 · 副标题 ———` 节奏栏，结构对齐
- **历史统计 6 张卡彩色色条**：紫 / 青 / 绿 / 黄 / 蓝 / 红 区分指标，数字用渐变文字
- **自动回复评估后主动点「继续沟通」**：AI 评分完毕（不论合不合适）会在简历面板里自动点一次 `btn-greet` 「继续沟通」打开输入框，再填草稿；低风险走 `dispatchEvent` 兜底，半 / 全自动走 `riskyClick`（启用 clicker 时是 OS 真鼠标）
- **半自动模式行为再明确**：⚡ 半自动 = 自动点「继续沟通」+ 自动填回复文字，**但不点发送**，留给人工核对后手动发；⚠️ 全自动 = 上面这些 + 自动点发送
- **待人工汇总卡片可点击跳转**：在「待人工处理汇总」里点任意一张卡，自动复制候选人姓名 + 切到 Boss 沟通页（粘贴搜索即可定位）
- **自动回复岗位范围卡折叠化**：模板 / 常用语 / 拒绝模板 / 冷却整组折叠，summary 实时显示当前选中的岗位
- **输入框聚焦发光环**：focus 时紫蓝色 3px ring，更易识别当前激活控件

### 修复

- **「继续沟通」按钮在低风险模式下不被点 → 草稿填不进输入框**：1.1.2 前低风险硬性跳过这次点击，导致输入框压根没出现；现在所有模式都执行（这颗按钮只是 UI 视图切换，不属于自动业务操作）
- **AI 评估抛异常时「继续沟通」步骤被跳过**：把点击挪到 try/catch 外层，无论评估成功失败都执行
- **空状态卡片不统一**：「暂无评分记录」/「当前没有待人工处理项」改成统一虚线 + 径向辉光样式

### 内部

- 引入 `--surface-0/1/2`、`--border-soft/strong`、`--accent/-2/-3` 等 CSS 设计变量，为后续主题切换打基础
- 新增 `ensureContinueCommunicationClicked()` / `isSemiAutoClickerMode()` / `findContinueCommunicationButton()` 三个 content 脚本 helper
- 新增 popup 侧 `attachManualReviewClickHandlers()` / `updateAutoReplyScopeHint()`

## [1.1.1] - 2026-05

首个公开开源版本。基于内部 1.0.4 低风险半自动版迭代而来。

### 新增

- **历史卡片点击 → Boss 中定位**：评分历史卡片可点击；自动切到对应岗位 + 推荐牛人页 + 滚到候选人卡片高亮显示
- **跨页定位 + 三级查找**：当前子页找不到时自动滚动加载更多 / 切换到推荐 · 精选 · 最新三个子页继续找
- **按岗位区分已筛选记录**：相同候选人在不同岗位下会被重新评分；同岗位下下次出现自动跳过，不重复调 AI
- **本地评分历史 30 天自动保留**：超期记录在下次运行时自动清理
- **CSV 按日期范围导出**：今日 / 近 7 天 / 近 30 天 / 全部 + 状态 + 关键词组合过滤
- **结论 + 推荐 / 淘汰理由分点**：历史卡片改成"结论 + 编号理由清单"格式
- **Gap 硬过滤**：候选人简历最新一段工作的结束日期距今超过 N 个月直接淘汰；写"至今"视为在职
- **AI 排除词语义扩展**：欧卡 / 写实 / 二次元 / Q 版 / 国风 / 外包 等会主动扩展为同义词族
- **大厂白名单大幅扩充**：覆盖互联网平台 + 国内外游戏厂商 + 动画影视 + 平台 IP
- **AI 防幻觉规则**：禁止 reason 里编造简历中不存在的项目名 / 公司名
- **节省 Token 监控**：每轮结束日志输出"跳过同岗位已评过的 N 人（节省 N 次 AI 调用）"
- **进阶运行设置 / 筛选条件 框中框分组**：相关项折叠分类，减少视觉冗余
- **历史页清爽**：自动隐藏运行日志框 + 开始/停止按钮组
- **视口自适应**：日志框 / 历史列表自动撑满浏览器窗口剩余高度，不外溢

### 修复

- 历史记录重复显示（cross-link 同记录写到多个 key 但展示侧未去重）
- AI 判定不通过的候选人未写入 storage 导致历史里看不到淘汰记录
- 候选人 ID cross-link 不彻底导致刷新页面后 AI 重复调用浪费 token
- SPA 路由切换不触发 content script 重新 init 导致定位失败

### 移除

- 内部开发文档 / 调试笔记 / .bak 文件
- 私有岗位预设（替换为示例预设）
