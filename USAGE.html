🚀 macOS 用户极速三步

  1. 解压本 zip 到一个固定位置（推荐 ~/Documents/，别留在下载文件夹）
  2. 双击 install.command，跟着终端中文提示走（约 2-3 分钟）
  3. 装好后 Chrome 右上角扩展图标点出来，配 AI Key + 选岗位 → 开始

详细说明 看这里。

🌐 Windows / Linux 用户

  1. 解压 zip
  2. 打开 chrome://extensions/，右上角打开「开发者模式」
  3. 点「加载已解压的扩展程序」→ 选 boss-assistant/ 子文件夹

不能用 Mac 后台点击器，建议只用 🛡 低风险 模式。详情 看这里。

⚠️ 风险提示
此工具违反 Boss 直聘服务条款，账号有被风控 / 封禁的风险。
强烈建议优先用 🛡 低风险模式（默认）；用工作号 / 新号尤其慎用。
